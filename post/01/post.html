[AWS] multipart/form-data → API Gateway → Lambda → S3

html form에서 post, multipart/form-data 형식으로 AWS의 API게이트웨이, 람다를 거쳐 최종적으로 S3에 이미지 파일을 업로드 하는 방법을 소개합니다. 단순한 업로드가 아니라 람다에서 대상 파일의 사이즈를 조정하여 업로드합니다. 람다함수는 파이썬으로 작성하였습니다. 파일 업로드 과정은 그림-1과 같습니다.
